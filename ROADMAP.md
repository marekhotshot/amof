# AMOF Roadmap

## Phase 1 – Foundation (current)
- Manifest (amof.yaml)
- CLI (sync, status, context)
- Guardrails
- Context engine basics
- Single-service validation

## Phase 2 – Orchestrator v1 (next)
- Planner agent
- Sandbox/worktree engine
- Subagent lifecycle orchestration
- Structured reporting
- PR automation

## Phase 3 – Memory Layer
- SpacetimeDB integration
- Memory-driven planning
- Preference learning

## Phase 4 – Multi-agent Scaling
- Parallel subagents across repos
- Batch multi-repo tasks
- Auto-resolve merge conflicts

## Phase 5 – Enterprise Layer
- Audit and compliance reporting
- RBAC/SSO
- Dashboards and UI hardening
- Enterprise deployment options (on-prem/K8s/serverless)
